# 手具セット管理アプリ 初めて使う人向けガイド

このガイドは、メンバーが自分の手具位置を入力し、管理者がプロジェクト・参加者・手具・統合表示・印刷を管理するための説明書です。

アプリ内の案内、動画台本、Webサイト用の説明に分けて使えるように、画面ごとの機能説明と実際の操作手順を分けています。

## まず知っておく言葉

| 言葉 | 意味 |
| --- | --- |
| プロジェクト | 1つの大会・作品・チーム単位の作業場所です。例: YOKOHAMA ROBINS 2026 |
| 招待ID | プロジェクトに参加するためのIDです。半角英数字・ハイフン・アンダーバー、3〜24文字で設定します。 |
| シート / バージョン | 県大会、関東大会、全国大会など、同じプロジェクト内で更新していくドリルシートです。 |
| 参加者 | 手具を配置するメンバーです。自分の名前を選んで入力します。 |
| 手具 | M1、ライフル、セイバーなど、ドリル上に置く丸の種類です。 |
| 入場 / 退場 | それぞれ別のシートとして手具位置を入力します。 |
| 統合表示 | 全員分の配置を重ねて確認する表示です。 |
| PWA | スマホのホーム画面に追加して、アプリのように使えるWebアプリです。 |

## 画面ごとの機能

### プロジェクト一覧

アプリを開いた最初の画面です。

- 既存のプロジェクトを選んで開けます。
- 右下の `+` から、プロジェクトの作成または参加ができます。
- 各プロジェクトには、シート数と招待IDが表示されます。

### プロジェクト追加

右下の `+` を押すと表示されます。

- `作成`: 新しいプロジェクトを作ります。
- `参加`: 招待IDを入力して、自分のプロジェクト一覧に追加します。
- 招待IDは自分で入力できます。
- 招待IDの制限は、半角英数字・ハイフン・アンダーバー、3〜24文字です。
- `自動` を押すと招待IDを自動作成できます。
- コピーアイコンで招待IDをコピーできます。

現在のPWA版では、招待IDでプロジェクト枠を追加できます。リアルタイム共同編集やホストのデータ取得は、Supabase同期を本格接続した後に有効化する想定です。

### メイン編集画面

プロジェクトを開いた後の画面です。

- 上部に、プロジェクト名 / シート名 / 編集中の参加者名が表示されます。
- 人型アイコンから参加者管理を開きます。
- PDFアイコンから印刷・PDF出力を開きます。
- 歯車アイコンからプロジェクト設定を開きます。
- `入場` / `退場` を切り替えて、それぞれの手具位置を入力します。
- `編集中` / `統合表示` を切り替えます。
- 参加者チップから、自分または編集したい参加者を選びます。
- `配置する手具` の `変更` から手具を選びます。

### ドリルシート

手具の丸を配置する場所です。

- タップ: 選択中の手具を配置します。
- 丸をドラッグ: 置いた丸を移動します。
- ピンチ操作: 拡大縮小します。
- `+` / `-`: 拡大縮小します。
- 丸を選択: 下に矢印キーが出ます。
- 矢印キー: 丸を1歩ずつ移動します。
- `0.5歩` に切り替え: 細かく調整します。
- 削除: 選択中の丸を削除します。

ドラッグしづらい場合は、丸を一度選択してから下の矢印キーで調整してください。

### 手具画面

メイン編集画面の `変更` から開きます。

- 手具をタップすると、次に配置する手具として選択されます。
- 目アイコンで、手具またはフォルダーの表示 / 非表示を切り替えます。
- フォルダーを開閉できます。
- フォルダーを追加できます。
- 手具を追加できます。
- 手具追加時に、名前・色・入れるフォルダーを設定できます。
- 色はプリセットから選ぶか、`+` から自由色を選べます。

### 参加者管理

メイン編集画面の人型アイコンから開きます。

- 招待IDと招待リンクを確認できます。
- 招待リンクをコピーできます。
- 参加者の名前を変更できます。
- 丸の中に表示する文字を編集できます。
- 参加者を追加できます。
- 統合表示に現在の参加者を追加できます。
- 全員を統合表示に追加できます。

参加者削除は現在の画面にはまだありません。追加する場合は、誤削除防止の確認を入れて実装する予定です。

### プロジェクト設定

メイン編集画面の歯車アイコンから開きます。

- プロジェクト一覧に戻れます。
- プロジェクト名を変更できます。
- シート / バージョン名を変更できます。
- シートを開けます。
- 現在のシートをコピーして、更新版を作成できます。
- 更新版シートを削除できます。
- PDF出力を開けます。
- 初期データに戻せます。

更新版シートを削除するときは、二段階確認が出ます。削除すると、そのシート上の丸の配置データは完全に失われます。元シートは削除できません。

### PDF出力

PDFアイコンまたはプロジェクト設定から開きます。

- 統合表示を出力できます。
- 参加者別のシートを出力できます。
- 入場だけ、退場だけ、両方を選べます。
- ドリルシート全体が入る形で保存します。

## メンバー向けの使い方

### 1. ホーム画面にアプリとして追加する

iPhone / iPad:

1. Safariで `https://prop-setting.vercel.app` を開きます。
2. 共有ボタンを押します。
3. `ホーム画面に追加` を押します。
4. 名前を確認して `追加` を押します。

Android:

1. Chromeで `https://prop-setting.vercel.app` を開きます。
2. 右上のメニューを押します。
3. `ホーム画面に追加` または `アプリをインストール` を押します。
4. 名前を確認して追加します。

iPhone / iPadでは、Androidのようなインストール案内ポップアップは基本的に自動では出ません。Safariの共有メニューから追加してください。

### 2. プロジェクトを作成または参加する

管理者から招待IDを受け取っている場合:

1. プロジェクト一覧の右下 `+` を押します。
2. `参加` を選びます。
3. 招待IDを入力します。
4. `参加` を押します。

自分で練習用のプロジェクトを作る場合:

1. プロジェクト一覧の右下 `+` を押します。
2. `作成` を選びます。
3. プロジェクト名を入力します。
4. 招待IDを入力するか、`自動` で作成します。
5. `作成` を押します。

### 3. プロジェクトを開く

プロジェクト一覧で、使いたいプロジェクトの `開く` を押します。

### 4. 自分の参加者を選ぶ

メイン画面の参加者一覧から、自分の名前を選びます。

名前がない場合は管理者に追加してもらうか、参加者管理から追加します。

### 5. 手具を選ぶ

1. `配置する手具` の `変更` を押します。
2. 配置したい手具を選びます。
3. 手具画面が閉じ、選んだ手具が配置対象になります。

### 6. ドリルに手具を置く

1. ドリルシート上の置きたい場所をタップします。
2. 丸が配置されます。
3. 位置を直したい場合は丸をドラッグします。
4. 拡大したい場合はピンチ操作、または `+` を使います。
5. ドラッグがやりにくい場合は、丸を選択して下の矢印キーで動かします。
6. 細かく調整したい場合は `0.5歩` に切り替えます。

### 7. 次の手具を選んで繰り返す

1. `変更` から次の手具を選びます。
2. ドリルに配置します。
3. 必要な手具が全部置けるまで繰り返します。

同じシート・同じ参加者・同じ入場/退場で、同じ手具をもう一度置くと、前の丸は置き換わります。

### 8. 入場と退場を両方入力する

1. `入場` を入力します。
2. `退場` に切り替えます。
3. 同じように手具を選んで配置します。

### 9. 完了

入場と退場の両方で必要な手具を置けたら完了です。管理者は統合表示やPDF出力で全体を確認できます。

## 管理者向けの使い方

### 1. プロジェクトを作成する

1. プロジェクト一覧の右下 `+` を押します。
2. `作成` を選びます。
3. プロジェクト名を入力します。
4. 招待IDを入力します。
5. 必要なら `自動` で招待IDを作成します。
6. コピーアイコンで招待IDをコピーします。
7. `作成` を押します。

招待IDは、半角英数字・ハイフン・アンダーバー、3〜24文字で設定します。例: `YR-2026`、`KANTO_2026`

### 2. 招待IDを共有する

1. プロジェクトを開きます。
2. 人型アイコンから参加者管理を開きます。
3. 招待IDまたは招待リンクをコピーします。
4. メンバーに共有します。

### 3. メンバーを設定する

1. 人型アイコンから参加者管理を開きます。
2. 参加者名を必要に応じて変更します。
3. 丸の中に表示する文字を確認します。
4. 必要な参加者を追加します。

現時点では参加者削除のUIはありません。削除機能を追加する場合は、シート削除と同じように確認を入れるのが安全です。

### 4. 手具を設定する

1. メイン画面の `配置する手具` の `変更` を押します。
2. 既存の手具を確認します。
3. 必要に応じてフォルダーを追加します。
4. 手具名を入力します。
5. 色を選びます。
6. 入れるフォルダーを選びます。
7. `手具を追加` を押します。

手具やフォルダーの目アイコンを切り替えると、ドリル上の表示 / 非表示を管理できます。

### 5. 統合表示を確認する

1. 各メンバーが自分の配置を入力します。
2. 管理者はメイン画面で `統合表示` に切り替えます。
3. 全員分の丸が重なって表示されます。
4. 必要に応じて参加者管理から `全員を統合` を押します。

統合表示は、全体のかぶりや手具の置き場所の確認に使います。

### 6. 印刷 / PDF出力をする

1. PDFアイコンを押します。
2. 出力対象を選びます。
3. `統合表示` または `参加者別` を選びます。
4. `入場`、`退場`、`両方` を選びます。
5. `保存` を押します。

印刷用には、ドリルシート全体が入るPDFとして保存します。

### 7. シートをバージョンアップする

大会ごとにシートを更新する場合は、コピー作成を使います。

例:

1. `県大会` シートを作る。
2. プロジェクト設定を開く。
3. コピー名に `関東大会` と入力する。
4. `コピー作成` を押す。
5. 関東大会用に丸の位置を修正する。
6. 次に必要なら `全国大会` をコピー作成する。

これにより、県大会、関東大会、全国大会のように履歴を残しながら更新できます。

不要になった更新版シートは削除できます。削除時は二段階確認があり、削除したデータは完全に失われます。

## よくある困りごと

### iPhone / iPadでアプリ追加の案内が出ない

iPhone / iPadでは、Androidのようなインストールポップアップは基本的に出ません。Safariで開き、共有ボタンから `ホーム画面に追加` を選んでください。

### 丸をドラッグしづらい

丸を一度タップして選択し、下に出る矢印キーで動かしてください。通常は1歩ずつ、`0.5歩` に切り替えると細かく動かせます。

### 拡大したらシートが動く

丸を直接ドラッグすると丸が動きます。シートを動かしたい場合は余白を動かす、または二本指操作を使ってください。細かい移動は矢印キーが安全です。

### 手具が表示されない

手具画面を開き、手具またはフォルダーの目アイコンが非表示になっていないか確認してください。

### 招待IDで参加しても他の人の編集が見えない

現在のPWA版では、招待ID付きのプロジェクトを自分の一覧に追加するところまで対応しています。リアルタイム共同編集はSupabase同期を接続してから有効化する想定です。

### データが消えないか不安

通常の編集内容は端末内に保存されます。ただし、更新版シートを削除すると、そのシートの丸の配置は完全に失われます。削除前には必ず二段階確認が出ます。

## アプリ内案内にするときの短い文言案

### 初回起動

手具セット管理へようこそ。まずはプロジェクトを作成するか、招待IDで参加してください。

### ホーム画面追加

iPhone / iPadはSafariの共有ボタンから `ホーム画面に追加` できます。AndroidはChromeのメニューから追加できます。

### 初めての配置

手具を選んで、ドリルをタップすると丸を置けます。置いた丸はドラッグ、または下の矢印キーで動かせます。

### 入場 / 退場

入場と退場は別々に入力します。上の切り替えから両方のシートを確認してください。

### 統合表示

統合表示では全員分の丸を重ねて確認できます。全体のかぶりや抜けをチェックしましょう。

## 動画にするときの流れ

### メンバー向け動画

1. ホーム画面に追加する。
2. 招待IDで参加する。
3. プロジェクトを開く。
4. 自分の名前を選ぶ。
5. 手具を選ぶ。
6. ドリルをタップして配置する。
7. ドラッグと矢印キーで調整する。
8. 入場 / 退場を切り替えて入力する。
9. 完了を管理者に伝える。

### 管理者向け動画

1. プロジェクトを作成する。
2. 招待IDをコピーして共有する。
3. 参加者名と丸内ラベルを整える。
4. 手具とフォルダーを整える。
5. 統合表示で全体を確認する。
6. PDF出力する。
7. 大会ごとにシートをコピー作成して更新する。
